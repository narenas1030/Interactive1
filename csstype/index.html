About the Type

Jeanne Type is a geometric typeface inspired by Josef Albers' modular type. It is also used by Ken Garland in his book Graphics Handbook and in Galt Toy's branding of the game Connect. The name Jeanne is a dedication to a special person that inspired me to become a better designer. Letters and Numericals are all set in lowercase.

I wanted to make the website to have a playful interaction that would make the typeface
pop up even more using different transform elements, colors, transitions, and hover states.
